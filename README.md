# hearthstone-like
a small game like hearthstone. on socket and jquery

#How to use
you need ```node js```
and then ```npm express```
and then ```npm socket.io```

the server wil be hosted on port 3000

#features
* login
* deck list
* deck generator
* card list
* card generator
* game queue
* the game
  * pick card
  * next turn
  * hand cards
  * table cards
  * enemy cards
  * life
  * round gems that increase at next turn
